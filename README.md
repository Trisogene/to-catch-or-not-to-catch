# Pokemon Shakespeare

A web app that shows Shakespearean descriptions of Pokemon.

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

## Test

```bash
pnpm test
```

## Features

- Search for any Pokemon by name
- View Pokemon description in Shakespearean English
- Add/remove Pokemon to favorites (persists after refresh)

## API

- [PokeAPI](https://pokeapi.co/docs/v2) - Pokemon data
- [Shakespeare Translator](https://funtranslations.com/api/shakespeare) - Translation
